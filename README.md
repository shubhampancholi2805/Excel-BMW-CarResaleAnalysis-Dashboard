# Excel-BMW-CarResaleAnalysis-Dashboard

## Project Overview
This project analyzes the **resale market of BMW vehicles** to understand the key factors affecting used car prices.  
The analysis focuses on identifying patterns between **price depreciation and variables such as age, mileage, engine size, fuel type, transmission, and model popularity**.

The goal of this project was to practice **data analysis, pivot tables, and dashboard building in Excel**, while extracting meaningful insights from the dataset.

---

## Objectives

The main objectives of this analysis were:

- Understand how **vehicle age affects resale price**
- Measure the impact of **mileage on car value**
- Identify **top-performing BMW models in the resale market**
- Analyze how **engine size influences price**
- Compare **fuel types and transmission types**
- Calculate the **average yearly depreciation rate**

---

## Dashboard Features

The final dashboard includes the following components:

### Key Performance Indicators (KPIs)

- Total Models
- Average Price
- Average Vehicle Age
- Average Mileage
- Top Performing Model
- Average Depreciation Rate per Year

### Visual Analysis

The dashboard visualizes the following relationships:

**Age vs Price**
- Shows how vehicle prices decline as cars get older.

**Mileage vs Price**
- Demonstrates the relationship between mileage and resale value.

**Top 5 BMW Models**
- Displays the most frequently appearing models in the resale market.

**Fuel Type Comparison**
- Compares average price and model distribution across fuel types.

**Engine Size vs Price**
- Shows how larger engines correlate with higher resale values.

**Transmission Distribution**
- Displays the proportion of Automatic, Manual, and Semi-Automatic vehicles.

### Interactive Filtering

A **Model slicer** allows users to filter the entire dashboard dynamically to analyze specific BMW models.

---

## Key Insights

### 1. Age is a Major Price Driver
Vehicle prices decline significantly during the **first few years**, after which the depreciation curve becomes flatter.

### 2. Mileage Strongly Influences Resale Value
Cars with **lower mileage maintain higher resale values**, while high-mileage vehicles experience significant price drops.

### 3. BMW 3 Series Dominates the Market
The **3 Series** appears as the most common model in the dataset, indicating high availability in the resale market.

### 4. Larger Engine Sizes Correlate with Higher Prices
Vehicles with **larger engine sizes tend to command higher resale prices**, reflecting performance-oriented models.

### 5. Semi-Automatic Transmission is Most Common
Among transmission types, **Semi-Automatic vehicles represent the largest share** of the dataset.
